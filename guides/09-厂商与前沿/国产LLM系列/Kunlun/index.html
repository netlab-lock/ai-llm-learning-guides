昆仑万维（Kunlun Tech）是中国领先的互联网与AI公司，市值约60亿美元。公司以游戏、社交平台起家，近年来全面转型为以AI为核心的科技集团，在大语言模型、AI音乐、AI办公等领域布局深远。

  • 成立时间2008年，北京
  • 创始人/CEO周亚辉
  • 上市代码深交所 300418
  • 估值/市值~$6B USD
  • 核心平台skywork.ai

Skywork 是昆仑万维打造的全能AI工作空间，集成5大Super Agent，覆盖文档、演示、表格、播客和网页搜索，一站式完成知识工作。

📄

Docs Agent

智能文档生成与编辑，支持长文写作、报告生成、多语言翻译。

📊

PPT Agent

AI驱动的演示文稿创建，自动生成幻灯片布局、图表与视觉设计。

📈

Excel Agent

智能表格处理，数据清洗、公式生成、可视化分析一气呵成。

🎙️

Podcast Agent

从文本自动生成播客节目，支持多角色对话、音色克隆。

🌐

Web Agent

联网搜索与信息整合，实时检索并总结网页内容。
