2.1 InternEvo：训练框架

InternEvo 是 InternLM 的分布式训练框架，特点：

  • 支持数据并行 (DP)、张量并行 (TP)、流水线并行 (PP)
  • 混合精度训练 (BF16/FP16)
  • 弹性训练，支持节点动态加入/退出
  • 与 DeepSpeed/FSDP 兼容

2.2 XTuner：微调工具

XTuner 是轻量级微调工具，支持：

  • LoRA：低秩适配，显存友好
  • QLoRA：4-bit 量化 + LoRA
  • 全参 SFT：全参数监督微调
  • 多种模型架构支持 (InternLM, LLaMA, Qwen 等)

2.3 LMDeploy：推理引擎

LMDeploy 是高效推理引擎，特性：

  • 量化：W4A16, W8A8 等多种量化方案
  • KV Cache 管理：PagedAttention
  • 持续批处理：continuous batching
  • TurboMind：高性能推理内核
  • 兼容 OpenAI API 格式

2.4 Lagent：智能体框架

Lagent 是 Agent 开发框架，支持：

  • 工具调用 (function calling)
  • 多步推理与规划
  • ReAct / Plan-and-Execute 等范式
  • 代码解释器、搜索、计算器等内置工具

2.5 OpenCompass：评测体系

OpenCompass 是国内最权威的大模型评测平台之一：

  • 100+ 评测 benchmark
  • 涵盖语言、推理、代码、数学、多模态等维度
  • 公正透明的评测方法论
  • 实时排行榜

3. InternVL：多模态系列

InternLM在3. InternVL：多模态系列方面的技术路线采用"视觉编码器+语言模型"的两阶段方案。视觉编码器（如ViT或SigLIP）将图像转换为token序列，然后与文本token一起输入语言模型。这种方案的数学本质是将不同模态的数据映射到统一的向量空间中。InternLM在这方面的创新包括动态分辨率处理、视觉token压缩和跨模态注意力优化。

InternVL多模态架构的技术演进与能力边界

InternVL系列从1.0到2.5的演进体现了视觉-语言模型在分辨率处理、多图理解和视频理解方面的持续突破。通过动态分辨率策略和跨模态注意力机制，InternVL在多个权威多模态基准上达到了开源模型最高水平，展现了上海AI实验室在多模态研究方面的深厚积累。

InternLM的多模态架构将视觉、音频等模态与语言模型深度融合。核心技术包括：模态特定编码器（将不同模态转换为统一向量表示）、跨模态注意力（实现模态间的信息交互）、以及统一的序列建模（在同一Transformer中处理所有模态）。这种设计使得模型能够进行跨模态推理。

多模态理解的关键挑战在于模态对齐——如何让视觉特征和文本特征在同一个语义空间中对齐。InternLM通过对比学习和交叉注意力机制，将不同模态的特征映射到共享的表示空间，使得模型能够理解图像中的文字、听懂语音中的含义。

import torch
import time

def benchmark(model, input_data, warmup=10, iters=100):
    for _ in range(warmup):
        model(input_data)
    torch.cuda.synchronize()
    start = time.perf_counter()
    for _ in range(iters):
        model(input_data)
    torch.cuda.synchronize()
    elapsed = (time.perf_counter() - start) / iters
    print(f"Latency: {elapsed*1000:.2f}ms")
    return elapsed
