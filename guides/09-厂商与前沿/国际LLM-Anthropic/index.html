章节导航

通俗类比：Claude就像AI中的"学者型选手"——特别擅长长文本分析和深度推理。它以安全性著称（Constitutional AI），就像给AI装了一套"道德准则"。长上下文能力（200K tokens）是它的杀手锏，能一次性读完整本小说并回答问题。
常见误区：关于本节的常见误区是只停留在理论层面。建议动手实验——用代码验证每个概念，而不仅是"觉得理解了"。很多细节只有在实际运行中才会暴露出来。
深入探讨：从更深层面看，本节涉及的技术选择往往是多维度权衡的结果——精度vs速度、延迟vs吞吐、成本vs效果。理解这些权衡比记住具体数值更重要，因为硬件和算法在快速演进，数值会变，但权衡的逻辑不变。

pip install anthropic

# 第一次调用
import anthropic
client = anthropic.Anthropic(api_key="sk-ant-...")
response = client.messages.create(
    model="claude-sonnet-4-20250514",
    max_tokens=1024,
    messages=[{"role": "user", "content": "你好，Claude！"}]
)
print(response.content[0].text)
