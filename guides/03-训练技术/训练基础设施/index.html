📋 学习路线

01 🏗️ 训练基础设施总览

LLM 训练的基础设施层次（硬件→网络→存储→调度→框架→应用），大规模训练的核心挑战。

总览必读

02 💾 Checkpoint 管理

Checkpoint 内容（权重+优化器+调度器），同步/异步保存，Sharded Checkpoint (DCP)，格式选择，频率策略。

核心存储

03 🛡️ 容错训练

故障率分析，故障类型（GPU/网络/存储/Straggler），检测机制，自动恢复流程，弹性训练。

核心可靠性

04 📊 训练日志与监控

三层指标（训练/框架/硬件），W&B 和 TensorBoard 实战，告警策略，训练曲线解读。

核心监控

05 🧪 实验管理

超参数搜索（Grid/Random/Bayesian），关键超参数范围，实验追踪，数据版本管理 (DVC)。

进阶实验

06 🖥️ 训练集群运维

SLURM 和 K8s 调度，存储架构（NVMe/NFS/Lustre/S3），NCCL 网络配置。

进阶运维

07 💰 训练成本优化

成本构成分析，Mixed Precision/梯度累积/Activation Checkpointing/序列打包，成本建模公式。

实战成本
