🛡️ 模型安全进阶

本课程深入探讨大语言模型面临的安全威胁与防御策略，涵盖攻击技术、数据安全、隐私保护和法规合规等关键领域。

🎯 课程目标

掌握LLM安全威胁全景，理解各类攻击原理与防御手段，建立系统化的安全思维框架。

📚 课程大纲

目录
  1. 对抗攻击总览 — LLM威胁模型与攻击分类
  2. Prompt Injection深度 — 直接与间接注入攻击
  3. 数据投毒与后门 — 训练阶段安全威胁
  4. 模型水印与版权 — 输出水印与知识产权保护
  5. 隐私保护技术 — 联邦学习与差分隐私
  6. AI法规与合规 — 全球AI监管框架
