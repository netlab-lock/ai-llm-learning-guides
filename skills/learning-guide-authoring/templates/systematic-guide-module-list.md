# Template: Systematic Technical Deep-Dive Guide Structure

Use this as a reference when the user asks for a "系统性专题" or "comprehensive guide" on a technical topic.

## Standard Module Structure (14-16 modules)

Phase 1: Foundation (Modules 01-03, cyan accent)
  01 - 核心概念与数学本质
  02 - 内存/资源模型与开销分析
  03 - 关键机制深度解析

Phase 2: Optimization (Modules 04-06, green accent)
  04 - 架构级优化
  05 - 量化技术
  06 - 驱逐与稀疏化

Phase 3: Distributed and Cluster (Modules 07-10, orange accent)
  07 - 分布式推理中的X
  08 - 分离式架构
  09 - 集群调度
  10 - 集合通信

Phase 4: Application and Practice (Modules 11-14, purple accent)
  11 - 应用场景专题
  12 - 框架实现对比
  13 - 工程调优实战
  14 - 前沿研究与未来方向

## Index Page Design

Hero section with stats, SVG learning path diagram, color-coded module cards, prerequisites, quick reference table, suggested reading order.

## Module Content Pattern

Each module (500-800 lines): opening motivation, 7-9 major sections, three-layer explanations (通俗类比 / 技术原理 / 数学本质), 3-8 SVG diagrams, comparison tables, code examples, summary, navigation links.
