## 三、学习路线建议

### 入门路线 (1-2周)
1. 深度学习经典模型 (感知机→Transformer)
2. LLM系统性指南 第1-5章 (基础→推理使用)
3. Scaling Law 基础

### 进阶路线 (2-4周)
1. LLM系统性指南 第6-10章 (优化→前沿)
2. MoE/注意力前沿/长上下文
3. 训练数据工程 + 分布式训练

### 专项深入 (按需)
- **推理方向**: 推理框架 + 编译优化 + 量化 + 推理服务架构
- **训练方向**: 分布式训练 + RLHF + 训练基础设施
- **应用方向**: RAG + Agent + 代码生成 + 多模态
- **硬件方向**: GPU架构 + 非NVIDIA加速器 + 集合通信
